Class: zbDateTime

zbDateTime(year, month, day, hour, minute, second)

Gregorian calendar date and time class. Below are some important limitations.
  1. Date ranges from year 1600 to 4000.
  2. Works with leap years.
  3. Includes time.
  4. Does not include leap seconds.
  5. No time zones or day light savings.
  6. No localization formatting.
  7. Is immutable. Once created it can not be changed.

Constructor

new zbDateTime(year, month, day, hour, minute, second)

Create a zbDateTime object with the given year, month, day, hour, minute and second.
Parameters:
Name Type Description
year number The year of the date time. Can be any year after 1600 AD and before 4000 AD.
month number The month value of the date time. This can be anything from 1=Jan to 12=Dec.
day number The day value of the date time. Values from 1 to 31 depending on the month and year given.
hour number The hour of the date time. Any value between 0 and 23.
minute number The minute of the date time. This can be 0 to 59.
second number The second of the date time. This can be 0 to 59.
