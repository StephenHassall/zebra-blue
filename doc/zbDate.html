Class: zbDate

zbDate(year, month, day)

Gregorian calendar date class. Below are some important limitations.
  1. Date ranges from year 1600 to 4000.
  2. Works with leap years.
  3. No time.
  4. No time zones or day light savings.
  5. No localization formatting.
  6. Is immutable. Once created it can not be changed.

Constructor

new zbDate(year, month, day)

Create a zbDate object with the given year, month and day.
Parameters:
Name Type Description
year number The year of the date. Can be any year after 1600 AD and before 4000 AD.
month number The month value of the date. This can be anything from 1=Jan to 12=Dec.
day number The day value of the date. Values from 1 to 31 depending on the month and year given.
