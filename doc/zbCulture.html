new zbCulture(code, monthList, shortMonthList, dayOfWeekList, shortDayOfWeekList, ordinalDayList, am, pm, longDateTime, shortDateTime, longDate, shortDate, longTime, shortTime, firstDayOfWeek)

Create a zbCulture object with locale related settings.
Parameters:
Name Type Description
code string The culture code (en-GB for example).
monthList Array.<string> A list of month names, from January to December.
shortMonthList Array.<string> A list of short month names, from Jan to Dec.
dayOfWeekList Array.<string> A list of days of the week names, from Sunday to Saturday.
shortDayOfWeekList Array.<string> A list of short of days of the week names. from Sun to Sat.
ordinalDayList Array.<string> A list of the st,nd,rd,th parts of the day of the month. Must be 31 items long and in lower case.
am string The AM text.
pm string The PM text.
longDateTime string The long date time format.
shortDateTime string The short date time format.
longDate string The long date format.
shortDate string The short date format.
longTime string The long time format.
shortTime string The short time format.
firstDayOfWeek zbDayOfWeek The first day of the week.
