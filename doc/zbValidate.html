Class: zbValidate

zbValidate

Validate different parameters and there ranges. All functions are static. They all throw an exception if something is not valid.

Methods

(static) isArrayOfStringInRange(value, min, max)

Is the given parameter a valid array of strings and is its length within the given range.
Parameters:
Name Type Description
value Array.<string> The parameter value to test.
min number The minimum range value. The value can be the same as min but not below it.
max number The maximum range value. The value can be the same as max but not above it.
